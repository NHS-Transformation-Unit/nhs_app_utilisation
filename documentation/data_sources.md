# Data Sources
***

This document provides the details of the data sources used in the analysis.

| **Data Item**           | **Description**                 | **Link**        |
|-------------------------|---------------------------------|-----------------|
| NHS App Management Information | Publicly available publication on logins to the NHS App each month | [July 2025 data](https://digital.nhs.uk/data-and-information/publications/statistical/nhs-app-statistics/july-2025) |
| GP Practice Registrations      | Number of patients registered at each GP practice                  | [June 2025 data](https://digital.nhs.uk/data-and-information/publications/statistical/patients-registered-at-a-gp-practice/july-2025) |
| Sub-ICB Locations Lookup       | Lookup of sub-ICB locations to ICB                                 | Available from [Open Geography Portal](https://geoportal.statistics.gov.uk/datasets/ons::sub-icb-locations-to-integrated-care-boards-to-nhser-april-2024-lookup-in-en/about) |
| ONS Mid-Year Population Estimates | Estimates of population estimates by age                        | [2023 estimates](https://www.ons.gov.uk/peoplepopulationandcommunity/populationandmigration/populationestimates/bulletins/populationestimatesforenglandandwales/mid2023) |
| NHS App utilisation demographics | Demographics of app utilisation for age and deprivation via FOI  | FOI data available in the `data` folder |
| Outpatient Attendance data       | 2023/24 Outpatient attendances broken down by age and deprivation | [2023/24 data](https://digital.nhs.uk/data-and-information/publications/statistical/hospital-outpatient-activity/2023-24) |
| Population by IMD decile         | Population estimates for England by IMD decile                   | Available from [ONS](https://www.ons.gov.uk/peoplepopulationandcommunity/populationandmigration/populationestimates/adhocs/13773populationsbyindexofmultipledeprivationimddecileenglandandwales2020) |
